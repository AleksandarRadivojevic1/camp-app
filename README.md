## Description
This project allows users to view and manage camping locations, including creating, editing and deleting entries.

## Getting Started

### Prerequisites
Make sure you have **Node.js** and **MongoDB** installed.

### Installation

#### Clone the repository
git clone https://github.com/AleksandarRadivojevic1/camp-app.git

#### Navigate into the project directory
cd camp-app

#### Install dependencies
npm install

#### Environment variables
Create a .env file for your environment variables (please follow env.example file).

## Running the app
cd into the project directory and run **node app.js**.
Then open your browser and go to **http://localhost:3000**
